Rent A Wheels Project

Project Overview

This project, "Rent A Wheels," was developed by a class group of 3 memebers. It was part of our Object-Oriented Programming (OOP) course. It is a desktop application that helps users rent vehicles through an intuitive and easy-to-use interface.

The project focuses on key OOP principles such as inheritance, polymorphism, and encapsulation to manage vehicles, customers, and rental transactions efficiently.

Technologies used: Java, JavaFX
